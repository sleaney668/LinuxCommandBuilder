• The mkdir command is is used to create new directories.

  • The useradd command is a low-level utility.
  • It is used for creating user accounts in Linux.
  • The adduser command can also be used, it is a symbolic link to useradd.
Note
  •  
  • useradd -d home/directory username will create a user with a home/directory.
  • useradd -g groupname username will create a new user and assign them a group.
  •  
  • The groupadd command is used in Linux to add user groups to a system.
Note
  • groupadd exits with the following value
  •  
  • 0   -   Success
  • 2   -   Invalid command syntax
  • 3   -   Invalid argument to option
  • 4   -   GID not unique
  • 9   -   Group name not unique
  • 10   -   Can't update group file
  • The rm command can be used to delete files.
  • It can also be used for deletion needs, see below.
Note
  • USE WITH CAUTION
  •  
  • rm -f will forcefully remove file.
  • rm -r will recursively remove files in the working directory.
  • rm -rf will remove the specified directory and all files and subdirectories from within.
  • rm -i will request confirmation for the deletion.
  • The rmdir command is used to delete empty directories.
Note
  •  
  • To delete non-empty directories use rm -r.

  • The mv command is used to move or rename files.

  • The vi command is used to edit the contents of files.
  • It is short for "vi"sual editor.
  • It has two modes COMMAND and INSERT mode:
    • Command mode: letters you type perform functions.
    • Insert mode: letters you type form words and sentences.
  • A more feature rich version of vi is vim, it stands for vi improved.
  • An easier alternative to vi or vim is nano, man nano.

  • vi will start in command mode, to enter insert mode type 'i'.
  • To quit insert mode type Ctrl+c or press the Esc key.
  • To undo any changes enter command mode and type 'u', to undo all changes type 'U'.
Note
  •  
  • Exit
  • :x        Exits vi, saving changes.
  • :q        Quits vi as long as no changes have been made, add ! to override.
  • :q!       Quits vi and ignores any changes.
  • :wq     Writes changes and quits vi.
  • ZZ       Exits vi, saving changes.

  • Arrow key or l      Navigate right.
  • Arrow key or k     Navigate up.
  • Arrow key or j      Navigate down.
  • H     Jump to top.
  • L     Jump to bottom.
  • /     Starts search, provide keyword. Use n to navigate through matches.

  • The passwd command is used to reset the password of a users account in linux.
Note
  • The following commands will change the way the passwd command operates.
  •  
  • -d username        Deletes a users password, this is a quick way to disable logins for an account (without disabling the account itself).
  • -e username        Immediately expire an accounts password, this forces a user to change their password the next time they login.
  • -h                            Display information about how to use the passwd command.
  • -i x username      This option is used to disable an account after the password has been expired for (x) number of days.
  • -l username         Lock the password of the named account, meaning they are not allowed to change their password.
  • -n x username     Set the minimum number of days (x) between password changes, 0 means the user may change their password at any time.
  • -q username        Quiet mode, will operate without displaying any output.
  • -S username        Displays account status information, specifying -a in addition to -S will display password status for all users.
  • -u username        Unlock the password of the named account.
  • -w x username     Set the number of days (x) of warning before a password change is required.
  • -x x username      Set the maximum number of days (x) a password remains valid
