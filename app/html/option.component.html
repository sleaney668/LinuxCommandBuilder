• -e           -     Select every process
  • -a           -     Select every process except session leaders
  • -f           -     Full output format
  • -r           -     Restrict the current running processes
  • -eaf       -     Combination of options
  • -aux       -     Combination of options
  • -i       -     List IP sockets
  • -i:     -     List IP sockets specifying port or port range e.g. 8080-8085
  • -u       -     List open files belonging to specified user
  • -p       -     List open files belonging to the specified process ID
  • +d       -     List open files belonging to the specified directory, does not descend into sub-directories
  • -t       -     List IDs of processes that have opened a particular file
